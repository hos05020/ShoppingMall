# ShoppingMall 
 ### Spring Boot와 Vue.js을 이용한 나만의 쇼핑몰 제작
 #### 사용 기술 및 데이터베이스(백엔드): Spring Boot,  QueryDSL, Spring Data JPA, MariaDB
 #### 사용 기술(프론트엔드): Vue.js
 ##### 간단한 CRUD을 기반으로한 사이트 입니다. -> 상품 등록, 상품 수정, 상품 목록, 특정 상품 내용을 확인 할 수 있습니다.
